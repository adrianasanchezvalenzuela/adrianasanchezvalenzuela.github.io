─── ⋆⋅☆⋅⋆ ──Introducción a los algoritmos y estructura de datos─── ⋆⋅☆⋅⋆ ──

Las estructuras de datos que veremos en el semestre son:

Algoritmo

Un algoritmo es un conjunto de instrucciones o reglas definidas y no-ambiguas, ordenadas y finitas que permite, típicamente, solucionar un problema, realizar un cómputo, procesar datos y llevar a cabo otras tareas o actividades.

Arreglo

Un arreglo es como una fila ordenada de casillas, donde cada casilla guarda un valor, como números o palabras, en un orden fijo.

Matrices

Una matriz es un conjunto de números o símbolos organizados en filas y columnas, de forma rectangular.

Pila

Se define formalmente como una colección de datos a los cuales se puede acceder mediante un extremo.

Cola

Estructura de datos que almacena elementos en una lista y permite acceder a ellos por uno de los dos extremos. Se caracteriza por seguir el principio FIFO (First In First Out), es decir, el primer elemento en entrar es el primero en salir.
